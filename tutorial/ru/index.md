# Самоучитель mclang

Этот самоучитель учит писать на mclang через живые примеры и задачи. Каждый урок строится на предыдущем.

Вам не нужно знать Си. Нужно уметь читать математические формулы — примерно на уровне школьной программы.

## Как работать с уроком

Каждый урок состоит из трёх частей:

**Теория** — объясняется концепция с примером кода.

**Практика** — вы пишете формулы сами и компилируете:
```bash
node dist/cli/index.js my_file.mc
gcc my_file.c src/main.c -lm -o demo && ./demo
```

**Задачи** — упражнения с нарастающей сложностью. Решения даются в конце урока.

## Перед началом
