# Урок 3 — LaTeX и неявное умножение

mclang понимает LaTeX-нотацию для дробей, корней, функций и греческих букв. Формулы из Wolfram Alpha, Wikipedia или учебника можно переносить почти буквально.

---

## 3.1 Дроби: `\frac{a}{b}`

В математике дробь $\frac{a}{b}$ — это деление. В mclang:

```mc
// Число π через формулу Лейбница (первые члены)
leibniz_approx() = 4.0 * (1.0 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \frac{1}{9})

// Среднее гармоническое двух чисел
harmonic_mean(a, b) = \frac{2*a*b}{a + b}

// Формула Пуазёйля: поток через трубу
poiseuille(r, l, ΔP, η) = \frac{π * r^4 * ΔP}{8 * η * l}
```

`\frac{числитель}{знаменатель}` раскрывается в `(числитель) / (знаменатель)` со скобками.

---

## 3.2 Корень: `\sqrt{x}` и `\sqrt[n]{x}`

```mc
// Квадратный корень
hypotenuse(a, b) = \sqrt{a^2 + b^2}

// Кубический корень стороны куба по объёму
cube_side(V) = \sqrt[3]{V}

// Общий корень n-й степени
nth_root(x, n) = \sqrt[n]{x}
```

`\sqrt[n]{x}` раскрывается в `pow(x, 1.0/n)`.

---

## 3.3 Греческие буквы

Греческие буквы — это обычные идентификаторы. В Си они транслитерируются:

| mclang | Си | Комментарий |
|--------|-----|-------------|
| `α` | `__uni_alpha` | строчная альфа |
| `β` | `__uni_beta` | строчная бета |
| `γ` | `__uni_gamma` | строчная гамма |
| `θ` | `__uni_theta` | тета (угол) |
| `λ` | `__uni_lambda` | лямбда (длина волны) |
| `μ` | `__uni_mu` | мю (среднее) |
| `σ` | `__uni_sigma` | сигма (стандартное отклонение) |
| `ω` | `__uni_omega` | омега (угловая частота) |
| `φ` | `M_PI * ...` нет — `__uni_phi` | фи (фазовый угол) |
| `π` | `M_PI` | константа π |

Вы можете использовать их и в параметрах, и в теле функции:

```mc
// Волновое уравнение: y = A·sin(kx - ωt + φ)
wave(A, k, x, ω, t, φ) = A * \sin{k*x - ω*t + φ}

// Закон Планка: спектральная яркость
planck(λ, T) =
    \frac{2*h*c^2}{λ^5} * \frac{1}{exp(\frac{h*c}{λ*k_B*T}) - 1.0}
```

Здесь `h`, `c`, `k_B` — встроенные физические константы (постоянная Планка, скорость света, постоянная Больцмана).

## 3.4 Неявное умножение

Когда число стоит вплотную перед идентификатором, скобкой или LaTeX-командой — это умножение:

```mc
// Все эти записи эквивалентны:
f1(r) = 2 * π * r
f2(r) = 2π * r
f3(r) = 2πr      // если нет пробела — πr сливается в один идентификатор!
```

**Важное правило:** пробел между символами — умножение. Отсутствие пробела — слияние в один идентификатор.

```mc
// 2π d → 2 * π * d (пробел разделяет π и d)
circumference(d) = 2π d

// 2πd → 2 * πd (πd — один идентификатор __uni_pid, не π * d!)
// Это почти наверняка не то, что вы хотите
```

Для LaTeX-функций пробел не нужен — они всегда разделяют:

```mc
// Всё это правильное неявное умножение:
f(x) = 2\sin{x}          // 2 * sin(x)
f(r, θ) = r\cos{θ}       // r * cos(θ)
f(x) = 3\sqrt{x}         // 3 * sqrt(x)
f(a, b) = 2\frac{a}{b}   // 2 * (a/b)
```

## 3.5 Реальная формула: нормальное распределение

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

```mc
normal_pdf(x, μ, σ) =
    \frac{1}{σ * \sqrt{2π}} * exp(-\frac{(x-μ)^2}{2σ^2})
```

Сравните с тем, как это выглядело бы в Си вручную:

```c
double normal_pdf(double x, double mu, double sigma) {
    return (1.0 / (sigma * sqrt(2.0 * M_PI))) *
           exp(-((x - mu) * (x - mu)) / (2.0 * sigma * sigma));
}
```

Версия mclang читается как оригинальная формула.

## 3.6 Степени через `^{...}`

Для составных показателей степени используйте фигурные скобки:

```mc
// e в степени -x²
gaussian(x) = e^{-x^2}

// Формула Эйлера: e^{iθ} — в mclang только вещественная часть
euler_real(θ) = e^{0} * \cos{θ}    // cos(θ), вещественная часть e^{iθ}

// Обратная связь в RC-цепи: u(t) = U₀ · e^{-t/(RC)}
rc_discharge(U0, R, C, t) = U0 * e^{-t / (R*C)}
```

## 3.7 Тригонометрия в LaTeX-стиле

Все стандартные функции доступны как `\sin{}`, `\cos{}`, `\tan{}` и т.д.:

```mc
// Закон Снеллиуса в красивом виде
snell(n1, θ1, n2) = \arcsin{\frac{n1 \sin{θ1}}{n2}}

// Формула де Муавра для cos(3θ)
cos3theta(θ) = 4\cos{θ}^3 - 3\cos{θ}

// Преобразование координат
polar_x(r, θ) = r\cos{θ}
polar_y(r, θ) = r\sin{θ}
```

Обратите внимание: `r\cos{θ}` — неявное умножение `r * cos(θ)`.

## 3.8 Полный пример: оптика

Скомбинируем всё в реальный модуль:

```mc
// Интенсивность двойной щели (опыт Юнга)
// δ — разность хода, λ — длина волны

young_intensity(I0, δ, λ) =
    I0 * \cos{\frac{π * δ}{λ}}^2

// Одиночная щель: интенсивность при дифракции
// a — ширина щели, θ — угол
single_slit(I0, a, λ, θ) =
    I0 * \left(\frac{\sin{β}}{β}\right)^2
    where
        β = \frac{π * a * \sin{θ}}{λ}
        β != 0    // особая точка при θ=0 обрабатывается отдельно

// Разрешающая способность (критерий Рэлея)
rayleigh_limit(λ, D) = 1.22 * \frac{λ}{D}
```

## Задачи

**Задача 1.** Запишите формулу силы электрического поля точечного заряда через `\frac`:

$$F = \frac{1}{4\pi\varepsilon_0} \cdot \frac{q_1 q_2}{r^2}$$

Используйте константу `ε₀ = 8.854e-12` как глобальную. Функция: `coulomb(q1, q2, r)`.

**Задача 2.** Формула Брэгга для дифракции рентгена:

$$n\lambda = 2d\sin\theta$$

Напишите функцию `bragg_lambda(n, d, θ)` — длина волны, при которой наблюдается дифракционный максимум n-го порядка. Используйте `\frac` и `\sin{}`.

**Задача 3.** Чёрное тело. Длина волны максимального излучения (закон Вина):

$$\lambda_{max} = \frac{b}{T}$$

где `b = 2.898e-3` м·К. Напишите `wien_peak(T)`. При какой температуре пик излучения приходится на видимый свет (λ ≈ 550 нм = 5.5e-7 м)?

**Задача 4 (★).** Полная формула Планка для спектральной яркости:

$$B(\lambda, T) = \frac{2hc^2}{\lambda^5} \cdot \frac{1}{e^{\frac{hc}{\lambda k_B T}} - 1}$$

Реализуйте `planck(λ, T)` используя встроенные константы `h` (постоянная Планка), `c` (скорость света), `k_B` (Больцмана). Разбейте на `numerator` и `denominator` в блоке `where`.

<summary>Показать решения</summary>

**Задача 1:**
```mc
ε0 = 8.854e-12

coulomb(q1, q2, r) = \frac{q1 * q2}{4π * ε0 * r^2}
```

**Задача 2:**
```mc
bragg_lambda(n, d, θ) = \frac{2 * d * \sin{θ}}{n}
```

**Задача 3:**
```mc
b_wien = 2.898e-3

wien_peak(T) = \frac{b_wien}{T}
```
Пик при 550 нм: `T = 2.898e-3 / 5.5e-7 ≈ 5269 К` — температура поверхности Солнца.

**Задача 4:**
```mc
planck(λ, T) =
    numerator / denominator
    where
        numerator   = 2.0 * h * c^2 / λ^5
        denominator = exp(\frac{h*c}{λ * k_B * T}) - 1.0
```

</details>
