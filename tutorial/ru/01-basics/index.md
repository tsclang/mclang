# Урок 1 — Первые формулы

В mclang каждый `.mc`-файл — это набор математических определений. Никаких классов, никаких `main()`, никакого `return`. Просто формулы.

---

## 1.1 Первая функция

Создайте файл `circle.mc`:

```mc
circle_area(r) = π * r^2
```

Скомпилируйте:

```bash
node dist/cli/index.js circle.mc
```

В той же папке появится `circle.c` и `circle.h`. Откройте `circle.c` — там будет:

```c
mc_num circle_area(mc_num r) {
    return (M_PI * pow(r, 2.0));
}
```

mclang заменил `π` на `M_PI`, а `r^2` на `pow(r, 2.0)`. Это и есть его работа — перевести математику в Си.

---

## 1.2 Несколько функций в одном файле

```mc
// Геометрия круга

circle_area(r) = π * r^2

circle_perimeter(r) = 2 * π * r

// Длина дуги при угле θ (в радианах)
arc_length(r, θ) = r * θ
```

Функции читаются сверху вниз. Каждая — отдельная строка-определение. Никаких скобок для группировки функций, никаких ключевых слов.

Проверьте формулу в `main.c`:

```c
#include <stdio.h>
#include "circle.h"

int main(void) {
    printf("Площадь круга r=5:      %.4f\n", circle_area(5.0));
    printf("Длина окружности r=5:   %.4f\n", circle_perimeter(5.0));
    printf("Полуокружность (π рад): %.4f\n", arc_length(5.0, 3.14159));
    return 0;
}
```

```bash
gcc circle.c main.c -lm -o demo
./demo
```

```
Площадь круга r=5:      78.5398
Длина окружности r=5:   31.4159
Полуокружность (π рад): 15.7080
```

## 1.3 Глобальные константы

Значения без параметров — это константы. Они разворачиваются во `inline`-подстановку в Си:

```mc
// Физические постоянные (SI)
k_spring = 200.0    // жёсткость пружины, Н/м
mass = 0.5          // масса, кг

// Угловая частота колебания пружины
omega(k, m) = sqrt(k / m)

// Частота нашей пружины
freq_spring = omega(k_spring, mass) / (2 * π)
```

`k_spring` и `mass` попадут в Си как `#define`-подобные подстановки. `freq_spring` — полноценная функция без параметров (вычисляется один раз при вызове).

Результат: `freq_spring() ≈ 3.183 Гц`.

---

## 1.4 Два параметра, многострочное выражение

Формулу можно разбить на строки — перенос работает, если строка заканчивается оператором:

```mc
// Расстояние между двумя точками на плоскости
dist2d(x1, y1, x2, y2) =
    sqrt(
        (x2 - x1)^2 +
        (y2 - y1)^2
    )
```

Открытая скобка `(` тоже разрешает перенос — строка внутри скобок может продолжаться сколько угодно.

## 1.5 Параметры по умолчанию

```mc
// Кинетическая энергия; по умолчанию считаем v в м/с, m в кг
kinetic(m, v = 1.0) = 0.5 * m * v^2
```

Вызов `kinetic(2.0)` использует `v = 1.0`. Вызов `kinetic(2.0, 10.0)` — `v = 10.0`.

В Си генерируется одна функция с двумя параметрами — дефолт применяется на стороне вызывающего кода.

## Задачи

**Задача 1.** Напишите функцию `rect_area(w, h)` — площадь прямоугольника, и `rect_diag(w, h)` — длина диагонали. Проверьте для `w=3, h=4` (ожидаемая диагональ: 5).

**Задача 2.** Напишите функцию `celsius_to_fahrenheit(c) = c * 9/5 + 32`. Проверьте: 0°C → 32°F, 100°C → 212°F.

**Задача 3.** Формула тела, брошенного горизонтально:

$$y(t) = h - \frac{g \cdot t^2}{2}$$

Напишите `fall_height(h, t)` с константой `g = 9.81`. С какой высоты нужно бросить тело, чтобы оно летело 3 секунды (y = 0)?

**Задача 4 (★).** Сила тяготения между двумя телами:

$$F = G \cdot \frac{m_1 \cdot m_2}{r^2}$$

Напишите функцию `gravity(m1, m2, r)`. Используйте встроенную физическую константу `G` (гравитационная постоянная). Вычислите силу притяжения Земли (5.972·10²⁴ кг) и тела массой 70 кг на расстоянии 6.371·10⁶ м.

---

## Решения

<details>
<summary>Показать решения</summary>

**Задача 1:**
```mc
rect_area(w, h) = w * h
rect_diag(w, h) = sqrt(w^2 + h^2)
```

**Задача 2:**
```mc
celsius_to_fahrenheit(c) = c * 9.0 / 5.0 + 32.0
```

**Задача 3:**
```mc
g = 9.81
fall_height(h, t) = h - g * t^2 / 2.0
```
Из `y = 0`: `h = g * t² / 2 = 9.81 * 9 / 2 ≈ 44.1 м`.

**Задача 4:**
```mc
gravity(m1, m2, r) = G * m1 * m2 / r^2
```
`G` — встроенная константа (6.674·10⁻¹¹). Результат: `≈ 685 Н` (примерно вес человека).

</details>
