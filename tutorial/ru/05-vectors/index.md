# Урок 5 — Векторы и физика

mclang поддерживает одномерные массивы (`num[]`) как векторы. Встроенные функции покрывают основные операции линейной алгебры; сложные вычисления можно строить поверх них.

---

## 5.1 Векторы как параметры

Тип `num[]` в объявлении функции означает «массив чисел произвольной длины». Компилятор автоматически добавляет скрытый параметр `_len`:

```mc
vec_sum(v) = \sum_{i=0}^{v.length-1} v[i]
```

В Си генерируется:

```c
mc_num vec_sum(const mc_num* v, int v_len) {
    mc_num _acc = 0.0;
    for (int i = 0; i <= v_len - 1; i++) _acc += v[i];
    return _acc;
}
```

Вам не нужно писать `v_len` в сигнатуре — он появляется сам.

---

## 5.2 Встроенные операции

```mc
// Скалярное произведение
dp = dot(u, v)

// Евклидова норма (длина вектора)
len = norm(v)

// Векторное произведение (только 3D)
n = cross(a, b)
```

Пример — единичный вектор (нормализация):

```mc
normalize(v) = v / norm(v)
```

Поэлементное деление вектора на скаляр — `v / scalar` — работает автоматически.

## 5.3 Физика: проекция и угол

Угол между двумя векторами:

$$\cos\theta = \frac{\vec{u} \cdot \vec{v}}{|\vec{u}||\vec{v}|}$$

```mc
vec_angle(u, v) = \arccos{\frac{dot(u, v)}{norm(u) * norm(v)}}
```

Проекция вектора `a` на вектор `b`:

$$\text{proj}_b a = \frac{\vec{a} \cdot \vec{b}}{|\vec{b}|^2} \vec{b}$$

В mclang скаляр из `dot` нельзя напрямую умножить на вектор через `*` внутри одной функции-одиночки — для этого нужна вспомогательная функция:

```mc
// Скалярный коэффициент проекции
proj_scalar(a, b) = \frac{dot(a, b)}{norm(b)^2}

// Полная проекция — b, умноженный на скаляр
// (поэлементно через вспомогательную функцию)
proj_vec(a, b) = proj_scalar(a, b) * b
```

## 5.4 Работа и мощность

Работа силы `F` вдоль перемещения `d`:

$$W = \vec{F} \cdot \vec{d}$$

```mc
work(F, d) = dot(F, d)

// Мощность = работа / время
power(F, d, t) = work(F, d) / t

// Мощность через скорость: P = F·v
power_inst(F, v) = dot(F, v)
```

Проверьте: сила `F = [10, 0, 0]` Н, перемещение `d = [5, 3, 0]` м.
`W = dot(F, d) = 10*5 + 0*3 + 0*0 = 50 Дж`.

## 5.5 Момент силы (торк)

$$\vec{\tau} = \vec{r} \times \vec{F}$$

```mc
torque(r, F) = cross(r, F)

// Модуль момента
torque_magnitude(r, F) = norm(cross(r, F))

// Момент через угол (|τ| = |r||F|sin θ)
torque_angle(r, F) =
    norm(r) * norm(F) * \sin{vec_angle(r, F)}
```

## 5.6 Кинетика твёрдого тела

Момент инерции однородного стержня длиной `L`, масса `m`, ось через центр:

$$I = \frac{1}{12} m L^2$$

```mc
inertia_rod_center(m, L) = \frac{1}{12} * m * L^2

inertia_rod_end(m, L) = \frac{1}{3} * m * L^2

// Теорема Штейнера: параллельный перенос оси
inertia_parallel(I_cm, m, d) = I_cm + m * d^2
```

Угловое ускорение при известном торке:

$$\alpha = \frac{\tau}{I}$$

```mc
angular_accel(τ, I) = τ / I
    where
        I > 0
```

## 5.7 Отражение и преломление (3D)

Отражение вектора `d` от поверхности с нормалью `n`:

$$\vec{r} = \vec{d} - 2(\vec{d} \cdot \hat{n})\hat{n}$$

```mc
reflect(d, n) = d - 2.0 * dot(d, n) * n
```

Преломление по закону Снеллиуса в векторной форме:

$$\vec{t} = \frac{n_1}{n_2}\vec{d} + \left(\frac{n_1}{n_2}\cos\theta_i - \cos\theta_t\right)\hat{n}$$

```mc
refract(d, n, n1, n2) =
    (n1/n2) * d + (n1/n2 * cos_i - cos_t) * n
    where
        cos_i = -dot(d, n)
        sin2t = (n1/n2)^2 * (1.0 - cos_i^2)
        cos_t = \sqrt{1.0 - sin2t}
        sin2t <= 1.0    // нет полного внутреннего отражения
```

## 5.8 Агрегаторы

Встроенные функции для статистики:

```mc
// Среднее
avg(v) = mean(v)

// Среднеквадратическое отклонение
deviation(v) = std(v)

// Нормализация вектора к диапазону [0, 1]
normalize_range(v) =
    (v - min_v) / (max_v - min_v)
    where
        min_v = min(v)
        max_v = max(v)
        max_v != min_v
```

## Задачи

**Задача 1.** Напишите `vec_dist(a, b)` — расстояние между двумя точками в пространстве (евклидово). Проверьте для `a=[0,0,0]` и `b=[3,4,0]` (ожидается 5).

**Задача 2.** Центр масс системы частиц. Для N частиц:

$$\vec{r}_{cm} = \frac{\sum_i m_i \vec{r}_i}{\sum_i m_i}$$

В mclang можно хранить только однородные массивы. Реализуйте `cm_x(masses, xs)` — x-координату центра масс через `dot(masses, xs) / sum(masses)`.

**Задача 3.** Угол падения луча. Дан вектор луча `d = [-1, -1, 0]` (нормированный) и нормаль к поверхности `n = [0, 1, 0]`. Угол падения — угол между лучом и нормалью. Используйте `vec_angle` из примера выше, но учтите: угол должен быть в `[0, π/2]`.

**Задача 4 (★).** Сила Кориолиса:

$$\vec{F}_{Cor} = -2m(\vec{\omega} \times \vec{v})$$

Напишите `coriolis(m, omega, v)` — вектор силы Кориолиса. Проверьте для `m=1`, `ω=[0,0,7.27e-5]` (вращение Земли), `v=[100,0,0]` м/с.

---

## Решения

<details>
<summary>Показать решения</summary>

**Задача 1:**
```mc
vec_dist(a, b) = norm(b - a)
```

**Задача 2:**
```mc
cm_x(masses, xs) = dot(masses, xs) / sum(masses)
// Аналогично cm_y и cm_z
```

**Задача 3:**
```mc
// Угол падения: d·n = cos(угол). Луч идёт "вниз", n — "вверх".
// dot([-1,-1,0]/√2, [0,1,0]) = -1/√2 → угол = 135°.
// Угол падения = π - угол = 45°.
incident_angle(d, n) = π - vec_angle(d, n)
```

**Задача 4:**
```mc
coriolis(m, omega, v) = -2.0 * m * cross(omega, v)
```
Результат: `F = [0, 0, -0.01454]` Н — сила перпендикулярна скорости, отклоняет вправо (в Северном полушарии).

</details>
