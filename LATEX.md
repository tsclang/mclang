# MClang — Поддерживаемый синтаксис LaTeX

Правило лексера: если токен начинается с `\`, слеш отбрасывается, идентификатор ищется в таблице ниже. Если совпадение найдено — применяется трансляция. Если нет — ошибка компиляции.

---

## 1. Арифметика

| LaTeX | Синоним без `\` | Трансляция в Си | Примечание |
|-------|----------------|----------------|-----------|
| `\frac{a}{b}` | — | `((a) / (b))` | Два блока `{}` обязательны |

| `\abs{x}` | `abs(x)`, `\|x\|` | `fabs(x)` | |
| `\lfloor x \rfloor` | `floor(x)` | `floor(x)` | Скобки как единый токен |
| `\lceil x \rceil` | `ceil(x)` | `ceil(x)` | Скобки как единый токен |

## 4. Суммы, произведения и агрегаторы

### 4.1 По диапазону индексов

| LaTeX | Синоним | Трансляция в Си |
|-------|---------|----------------|
| `\sum_{i=a}^{b} expr` | `∑(i=a, b) expr` | `for`-цикл с аккумулятором |
| `\prod_{i=a}^{b} expr` | `∏{i=a}{b} expr` | `for`-цикл с аккумулятором |

```c
mc_num _acc = 0;   // для \sum; 1.0 для \prod
for (int i = a; i <= b; i++) _acc += expr;
```

### 4.2 По элементам массива (`\in`)

Парсер различает два варианта по наличию `\in` vs `=` в нижнем индексе.

| LaTeX | Синоним | Трансляция в Си |
|-------|---------|----------------|
| `\sum_{x \in v} expr` | `sum(v)` (если expr=x) | O(n) цикл по элементам |
| `\prod_{x \in v} expr` | `product(v)` (если expr=x) | O(n) цикл по элементам |
| `\min_{x \in v} expr` | `min(v)` (если expr=x) | O(n) цикл, возвращает минимум |
| `\max_{x \in v} expr` | `max(v)` (если expr=x) | O(n) цикл, возвращает максимум |
| `\inf_{x \in v} expr` | синоним `\min_{x \in v}` | то же, что `\min` для конечных множеств |
| `\sup_{x \in v} expr` | синоним `\max_{x \in v}` | то же, что `\max` для конечных множеств |

```c
// \sum_{x \in v} x^2
mc_num _acc = 0;
for (int _i = 0; _i < v_len; _i++) {
    mc_num x = v[_i];
    _acc += pow(x, 2);
}
```

### 4.3 Функции-агрегаторы для массивов

Все O(n). Dispatch по числу аргументов: один массив → агрегатор; два скаляра → попарная операция.

| Функция | Описание | Два скаляра |
|---------|---------|------------|
| `sum(v)` | сумма элементов | — |
| `product(v)` | произведение элементов | — |
| `mean(v)` / `\bar{v}` | среднее значение | — |
| `std(v)` / `\sigma{v}` | стандартное отклонение | — |
| `min(v)` | минимальный элемент | `min(a,b)` → `fmin(a,b)` |
| `max(v)` | максимальный элемент | `max(a,b)` → `fmax(a,b)` |

## 5. Математический анализ

| LaTeX | Трансляция | Метод |
|-------|-----------|-------|
| `\lim{x \to a} expr` | вычисление в `a + 1e-9` | численный |
| `\lim{x \to \infty} expr` | вычисление при `x = 1e15` | численный |
| `\frac{d}{dx} f(x)` | `(f(x+1e-7) - f(x)) / 1e-7` | конечные разности |
| `\int{a}{b} f(x) dx` | симметричная форма mclang | метод Симпсона |
| `\int_{a}^{b} f(x) dx` | стандартная LaTeX-форма | метод Симпсона |

### Синтаксис интеграла

Два поддерживаемых варианта:
```
\int{нижняя}{верхняя} выражение d переменная     // mclang-форма
\int_{нижняя}^{верхняя} выражение d переменная   // стандартная LaTeX
```

Примеры:
```
area = \int{0}{π} sin(x) dx
area = \int_{0}^{π} sin(x) dx
```

### Синтаксис предела

```
\lim{переменная \to значение} выражение
```

Пример:
```
L = \lim{x \to 0} \frac{sin(x)}{x}
```

### Синтаксис производной

```
\frac{d}{d переменная} выражение
```

Пример:
```
slope = \frac{d}{dx} x^2 + 2x
```

## 9. Автоматически масштабируемые скобки (`\left`/`\right`)

В LaTeX `\left(` и `\right)` задают автоматический размер скобок. В mclang размер игнорируется, остаётся только тип скобки:

| LaTeX | Трансляция |
|-------|-----------|
| `\left( expr \right)` | `(expr)` |
| `\left[ expr \right]` | `[expr]` |
| `\left\| expr \right\|` | `\|expr\|` → `fabs(expr)` |
| `\left. expr \right.` | `expr` (null delimiter) |

## 11. Правила парсинга LaTeX-блоков

### Аргументы в фигурных скобках `{}`

Аргумент `{}` — это произвольное выражение mclang, ограниченное фигурными скобками.

| Конструкция | Число аргументов `{}` |
|-------------|----------------------|
| `\frac{a}{b}` | 2 |
| `\sqrt{x}` | 1 |
| `\sqrt[n]{x}` | 1 (+ опциональный `[n]`) |
| `\sin{x}` | 1 |
| `\log_{b}{x}` | 2 (нижний индекс + аргумент) |
| `\sum_{i=a}^{b}` | нижний индекс `_{}` + верхний `^{}` |
| `\int{a}{b}` | 2 |
| `\lim{x \to a}` | 1 (содержит `\to`) |
| `\frac{d}{dx}` | 2 (производная: `d` и `dx`) |

### Скобки `{}` vs отсутствие скобок

- `x^2` — степень одного символа, скобки не нужны
- `x^{a+b}` — степень выражения, скобки обязательны
- `\sin x` — аргумент одного символа/числа без скобок: `sin(x)` ✓
- `\sin x + y` — применяется только к `x`: `sin(x) + y` ✓
- `\sin{x+1}` — аргумент выражения, скобки обязательны

### Нижний (`_`) и верхний (`^`) индексы в `\sum` / `\prod`

```
\sum_{i=0}^{n} expr
```

- `_{...}` — нижний индекс: инициализация счётчика `i=0`
- `^{...}` — верхний индекс: граница включительно `i <= n`

## 12.2 Кусочные функции (`\begin{cases}`)

Альтернативный синтаксис для `if/else`-цепочек:

```
f(x) = \begin{cases}
    x^2    & x < 0 \\
    0      & x == 0 \\
    \sqrt{x} & x > 0
\end{cases}
```

Правила:
- Каждый случай: `выражение & условие \\`
- Последний случай может быть без условия (`& \text{otherwise}` или просто без `&`) — это ветка `else`
- Разворачивается в цепочку `if/else` в Си

```c
if (x < 0)   return pow(x, 2);
if (x == 0)  return 0;
return sqrt(x);
```

## 14. Не поддерживается (v2+)

| LaTeX | Причина |
|-------|---------|
| `\int f dx` (неопределённый) | требует CAS |
| `\partial` (частная производная) | требует CAS |
| `\nabla` (градиент) | требует векторного анализа |
| `\oint` (контурный интеграл) | требует CAS |
| `\matrix`, `\pmatrix` | заменяется синтаксисом `[[...]]` |
| `\partial`, `\nabla` | требует CAS |
