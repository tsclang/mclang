# MClang MVP

## Входит в MVP

### Компилятор

- Лексер с поддержкой Unicode-идентификаторов и греческих букв
- Парсер (PEG, recursive descent)
- Построение AST
- Генератор Си-кода (`.c` + `.h`)
- Таргеты: `--target c`, `--target wasm`, `--target shared`, `--target rust`, `--target node`
- Флаг точности: `--precision f64` (default), `--precision f32`, `--precision fixed`

### Типы

- `num`, `int`
- `num[]` (вектор) + неявный `_len` параметр
- `num[][]` (матрица) + неявные `_rows`, `_cols` параметры
- Статические массивы `num[N]` → стек

### Синтаксис

- Объявление функций: `f(x) = expr`
- Глобальные константы: `G = 6.67e-11`
- Значимые отступы
- ASI (автоматические точки с запятой)
- Перенос строк через оператор или открытую скобку
- Параметры по умолчанию: `f(x, y = 0)`
- Неявный `return` (последнее выражение)
- `_` префикс → приватная функция
- Все функции экспортируются по умолчанию
- Комментарии `//` и `#`
- Импорт пользовательских `.mc`-файлов: `import "./file.mc"`, `from "./file.mc" import name`, `import "./file.mc" as alias`

### Математика

- Неявное умножение: `2x`, `2(a+b)`
- LaTeX: `\frac{a}{b}`, `\sqrt{x}`, `\sqrt[n]{x}`, `x^{n}`
- Степень: `x^n`, `x**n` → `pow(x, n)`
- Все тригонометрические функции + русские синонимы (`tg`, `ctg`, `sh`, `ch`, …)
- Логарифмы: `ln`, `log`, `log10`, `\lg`, `log{base}{x}`
- `exp(x)`, `e^x`, `abs(x)`, `|x|`, `\abs{x}`, `floor`, `ceil`, `round`
- Встроенные константы: `π`, `e`, `τ`, `φ`, `inf`, `nan`
- Физические константы: `G`, `c`, `h`, `k_B`, `N_A`, `R`

### Операторы

- Арифметика: `+`, `-`, `*`, `⋅`, `/`, `÷`, `%`, `.*`, `^`, `**`
- Сравнение: `<`, `>`, `<=`, `>=`, `==`, `!=`, `<>`, `≤`, `≥`
- Цепочки сравнений: `0 < x < 10`
- Логика: `and`, `or`, `xor`, `not`, `&&`, `||`, `!`, `∧`, `∨`, `⊕`, `¬`
- Принадлежность: `∈`, `∉`, `in`, `!in` + диапазоны `[a,b]`, `(a,b)` + множества `ℕ`, `ℤ`, `ℝ`
- Остаток: `%`, `mod(x, y)`
- Dispatch `⋅` по типу: скаляры → `*`, векторы → dot product, матрицы → матричное умножение
- Модуль `|x|` с контекстным разбором (`||` vs `|...|`)

### Управляющие конструкции

- `if / else` (инлайн + блочный), только `if-first`
- Guard clause: `if (cond) value` без `else`
- `for i in a..b`, `for i in a..b step n`
- `while expr`
- `∑`, `∏` (разворачиваются в `for`-цикл)

### Блок `where`

- Определения переменных (топологическая сортировка)
- Guard-и (порядок объявления, нарушение → `nan`)

### Векторы и матрицы

- Литералы: `[1, 2, 3]`, `[[1, 0], [0, 1]]`
- Поэлементные операции: `+`, `-`, `*` (на скаляр), `.*`
- Матричное умножение: `A ⋅ B`
- `dot(v1, v2)`, `cross(v1, v2)`
- Индексация: `v[i]`, `m[i][j]`, срезы `v[a..b]`, `m[:, j]`
- Функции: `transpose`, `det`, `inv`, `norm`
- Константы: `I(n)`, `zeros(r,c)`, `ones(r,c)`
- `.length`, `.rows`, `.cols`

### Специальные состояния

- `inf`, `-inf`, `nan`
- `is_nan(x)`, `is_inf(x)`, `is_finite(x)`
- Деление на ноль → `inf`/`nan`, не падение

### Сложные операции (численные методы)

- Предел `\lim{x \to a}` → вычисление в точке `a ± 1e-9`
- Производная `\frac{d}{dx}` → разностная схема `(f(x+h) - f(x)) / h`
- Интеграл `\int{a}{b}` → метод Симпсона через указатель на функцию
- `solve(t, lo, hi) { expr == val }` → метод бисекции, точность `1e-9`
- Рекурсия (взаимная и хвостовая)

### CLI

- `mclang eval <file.mc>` — список экспортируемых функций
- `mclang eval <file.mc> "fn(arg1, arg2)"` — вызов функции с аргументами (требует GCC в PATH)

## Не входит в MVP (v2+)

### CAS-движок

- Аналитические интегралы (таблица первообразных)
- Аналитические производные (таблица правил)
- Аналитические пределы
- Алгебраические упрощения (`sin²+cos²=1` и т.д.)
- Constant folding для сложных выражений (только тривиальный в MVP)
- Аналитический solve (символьное решение уравнений) и from-синтаксис в CAS

### Таблицы

- Числовые таблицы (массивы пар) с линейной интерполяцией
- Именованные таблицы со строковыми ключами (`["Al": 2700]`)
- `import_table("file.csv")` — compile-time импорт из CSV

### Стандартная библиотека

- `std.physics`, `std.chemistry`, `std.geom.*` — только интерфейс; данные и реализация в v2

### Оптимизации

- SIMD (SSE/AVX) авто-векторизация
- Оптимизация `dist(A,B)^2` → убирает `sqrt`

### Система единиц (Units)

- Аннотации `kg`, `m`, `s` и проверка совместимости
- Вывод производных единиц

### Модули

- `std.*` импорты с реальными данными

### Визуальный конструктор (v2)

Веб-приложение поверх рабочего компилятора. Требует стабильного Wasm-таргета.

**Редактор:**
- Рендеринг формул в реальном времени (LaTeX → MathML/Canvas)
- Палитра символов: греческие буквы, `∫`, `∑`, `∏`, `√`, `±`, …
- Автодополнение функций и констант

**Библиотеки:**
- Браузер модулей: `std.physics`, `std.chemistry`, `std.geom.*`
- Подключение одним кликом, автоимпорт

**Запуск:**
- Тестирование функций: задать аргументы → получить результат прямо в браузере (через Wasm)
- Симуляции: пошаговый запуск с визуализацией промежуточных значений
- Вывод результатов в таблицу

**Графики:**
- 2D-графики функций (y = f(x))
- Параметрические кривые
- Фазовые портреты для симуляций

**Визуальные блоки:**
- `if / else` — блок-схема
- `for`, `while` — визуальный цикл с диапазоном
- `where` — панель зависимостей и guard-ов
- `from "./file.mc" import` — дерево импортов

### Прочее

- `solve` без явного диапазона (дефолтный `[0, 1e9]`)
- Именованные таблицы с строковыми ключами (`"Al": 2700`)
- Comprehensions (`[[1 if i==j else 0 for i in 0..2] for j in 0..2]`)
- Научные префиксы (`10k`, `5m`, `2u`)
