# MClang — Лог реализации

**Язык реализации:** TypeScript (Node.js)

## Формат записи

```
### [YYYY-MM-DD] Фаза N — Название
**Статус:** в процессе / готово
- [ ] задача
- [x] выполненная задача
**Заметки:** ...
```
