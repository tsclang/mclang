# trunc — усечение к нулю

---

## Сигнатура

```mc
trunc(x: num) → num
```

Отбрасывает дробную часть в сторону нуля.

---

## Пример

```mc
f(x) = trunc(x)
```

```c
mc_num f(mc_num x) { return trunc(x); }
```

---

## Сравнение функций округления

| Функция | 2.7 | -2.7 |
|---------|-----|------|
| `floor(x)` | 2 | -3 |
| `ceil(x)` | 3 | -2 |
| `round(x)` | 3 | -3 |
| `trunc(x)` | 2 | -2 |

`trunc` всегда округляет **к нулю** — в отличие от `floor`, который всегда вниз.
