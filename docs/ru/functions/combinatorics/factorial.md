# factorial — факториал

---

## Сигнатура

```mc
factorial(n: int) → num
n!                       // оператор-синоним
```

---

## Пример

```mc
f(n) = factorial(n)
g(n) = n!               // эквивалентно
```

---

## Ограничения

- `n ∈ [0, 20]` — таблица
- `n > 20` → `INFINITY`
- `n < 0` → `NAN`
