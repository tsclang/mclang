# \lim — предел

---

## Синтаксис

```mc
\lim{x \to a} f(x)
```

---

## Метод

Вычисление в точке `a ± 1e-9` (численное приближение):

```c
// Генерируется:
mc_num _x_val = a;
mc_num _lim_result = f(_x_val + 1e-9);  // или среднее двух сторон
```

---

## Пример

```mc
// lim_{x→0} sin(x)/x = 1
sinc(x) = \lim{x \to 0} sin(x) / x   // → 1.0

// lim_{x→∞} 1/x = 0
inf_limit = \lim{x \to inf} 1/x
```

---

## Ограничения

- Только численный предел — не аналитический
- При разрывных функциях результат зависит от стороны приближения
- При `a = inf` приближается к `1e15`

---

## Применение

```mc
// Производная через предел (иллюстративно, используй \frac{d}{dx} вместо)
deriv_def(f, x) = \lim{h \to 0} (f(x + h) - f(x)) / h
```
