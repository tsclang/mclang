# \frac{d}{dx} — числовая производная

---

## Синтаксис

```mc
\frac{d}{dx} f(x)
```

---

## Метод

Разностная схема первого порядка: `(f(x+h) - f(x)) / h`, где `h = 1e-7`.

---

## Пример

```mc
// Производная sin(x) = cos(x)
dsin_dx(x) = \frac{d}{dx} sin(x)
// dsin_dx(0) ≈ 1.0
// dsin_dx(π/2) ≈ 0.0

// Производная x^2 = 2x
dparab(x) = \frac{d}{dx} x^2
// dparab(3) ≈ 6.0
```

```c
// Генерируется:
mc_num _h = 1e-7;
mc_num _deriv = (sin(x + _h) - sin(x)) / _h;
```

---

## Применение

```mc
// Градиент функции (по x)
grad_x(f, x, y) = \frac{d}{dx} f(x, y)

// Метод Ньютона
newton_step(f, x) =
    x - f(x) / \frac{d}{dx} f(x)
```

---

## Ограничения

- Числовая погрешность `~h = 1e-7` (не аналитическая производная)
- Для лучшей точности используй центральную разность: `(f(x+h) - f(x-h)) / (2h)` — реализуй вручную
- Медленно при многократном вызове
