# \int — числовое интегрирование

---

## Синтаксис

```mc
\int{a}{b} f(x) dx
```

---

## Метод

Метод Симпсона (составной), шаг `h = (b-a)/1000`.

---

## Пример

```mc
// Площадь под параболой
area = \int{0}{1} x^2 dx   // → 1/3 ≈ 0.3333

// Через функцию
arc_length(a, b) = \int{a}{b} sqrt(1 + cos(x)^2) dx
```

```c
// Генерируется вызов процедуры Симпсона:
mc_num _integral_result = mc_simpson(/* func_ptr, a, b, steps */);
```

---

## Ограничения

- Числовой метод — не аналитический
- Точность ~1e-9 при гладкой функции
- Медленно для сингулярных подынтегральных функций
- `dx` — обязательная часть синтаксиса

---

## Производительность

Интегрирование выполняется в **runtime**, не compile time. Если аргументы — константы, результат всё равно вычисляется при каждом вызове.

---

## Пример: момент инерции

```mc
inertia(m, R) = m * \int{0}{R} r^2 dr / R
```
