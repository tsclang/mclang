# cbrt — кубический корень

---

## Сигнатура

```mc
cbrt(x: num) → num
```

Область: ℝ (работает для отрицательных чисел, в отличие от `sqrt`).

---

## Пример

```mc
f(x) = cbrt(x)
```

```c
mc_num f(mc_num x) { return cbrt(x); }
```

---

## Отличие от x^(1/3)

`cbrt(-8) = -2`, тогда как `(-8)^(1/3)` в C даёт `nan` (дробная степень отрицательного числа через `pow`).

---

## Применение

```mc
// Сторона куба по объёму
side(volume) = cbrt(volume)

// Корень n=3 через cbrt точнее, чем sqrt(n, x)
```
