# sec — секанс

---

## Сигнатура

```mc
sec(x: num) → num
```

Область: ℝ \ {π/2 + πk}, значения: (-∞, -1] ∪ [1, +∞).

---

## Генерируемый C-код

```mc
f(x) = sec(x)
```

```c
mc_num f(mc_num x) { return (1.0/cos(x)); }
```

---

## Особые значения

| x | sec(x) |
|---|--------|
| 0 | 1 |
| π/3 | 2 |
| π/2 | ±∞ |
| π | -1 |
