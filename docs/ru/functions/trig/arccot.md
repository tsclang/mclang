# arccot — арккотангенс

---

## Сигнатура

```mc
arccot(x: num) → num
```

Реализован через `atan2(1, x)` — корректно обрабатывает `x = 0` (возвращает `π/2`).

---

## Пример

```mc
f(x) = arccot(x)
```

```c
mc_num f(mc_num x) { return atan2(1.0, x); }
```

---

## Область и значения

- Область: ℝ
- Значения: `(0, π)`
