# csc — косеканс

---

## Сигнатура

```mc
csc(x: num) → num
\csc{x}            // LaTeX
```

Область: ℝ \ {πk}, значения: (-∞, -1] ∪ [1, +∞).

---

## Генерируемый C-код

```mc
f(x) = csc(x)
```

```c
mc_num f(mc_num x) { return (1.0/sin(x)); }
```

---

## Особые значения

| x | csc(x) |
|---|--------|
| π/2 | 1 |
| π/6 | 2 |
| 0 | ±∞ |
| π | ±∞ |
