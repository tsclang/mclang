# erfc — дополнительная функция ошибок

---

## Сигнатура

```mc
erfc(x: num) → num
```

`erfc(x) = 1 - erf(x)`. Область: ℝ, значения: `(0, 2)`.

---

## Пример

```mc
f(x) = erfc(x)
```

```c
mc_num f(mc_num x) { return erfc(x); }
```

---

## Когда использовать вместо erf

При больших `x` значение `erf(x)` близко к 1, и `1 - erf(x)` теряет точность из-за вычитания близких чисел. `erfc` вычисляется напрямую без потери значимости.

```mc
// Правый хвост нормального распределения — используй erfc, не 1 - erf
tail_prob(x) = erfc(x / sqrt(2)) / 2
```
