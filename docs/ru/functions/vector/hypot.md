# hypot — гипотенуза / евклидово расстояние

---

## Сигнатура

```mc
hypot(x: num, y: num) → num
```

Вычисляет `√(x² + y²)` без промежуточного переполнения.

---

## Генерируемый C-код

```mc
f(x, y) = hypot(x, y)
```

```c
mc_num f(mc_num x, mc_num y) { return hypot(x, y); }
```

---

## Применение

```mc
// Длина вектора (x, y)
length(x, y) = hypot(x, y)

// Расстояние между двумя точками
dist(x1, y1, x2, y2) = hypot(x2 - x1, y2 - y1)

// Нормировка
normalize_x(vx, vy) = vx / hypot(vx, vy)
```

---

## Отличие от sqrt(x^2 + y^2)

`hypot` устойчив к переполнению: при больших `x`, `y` промежуточные квадраты могут выйти за `DBL_MAX`. C99 `hypot()` обходит это.
