# \phi — φ (золотое сечение)

---

## Синтаксис

```latex
\phi
φ      // Unicode
phi    // словесный синоним
```

---

## Трансляция

```c
((1.0 + sqrt(5.0)) / 2.0)   // ≈ 1.6180339887498949
```

---

## Примеры

```mc
golden_rect(a) = a * \phi
fibonacci_approx(n) = \phi^n / sqrt(5)
```
