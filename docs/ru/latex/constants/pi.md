# \pi — число π

---

## Синтаксис

```latex
\pi
π     // Unicode
pi    // словесный синоним
```

---

## Трансляция

```c
M_PI   // = 3.14159265358979323846
```

---

## Примеры

```mc
circle_area(r) = \pi * r^2
f(x) = sin(\pi * x)
TWO_PI = 2\pi           // неявное умножение
```
