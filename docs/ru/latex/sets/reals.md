# \mathbb{R} — вещественные числа

---

## Синтаксис

```latex
x \in \mathbb{R}
x \in ℝ
```

---

## Трансляция

```c
isfinite(x)
```

---

## Пример

```mc
is_real(x) = x ∈ ℝ   // исключает nan и inf
```
