# \in ∈

---

## Синтаксис

```latex
x \in S
x \in [a, b]
x \in \mathbb{N}
```

---

## Трансляция

```mc
x \in [0, 1]   →   0 <= x and x <= 1
x \in \mathbb{N}  →  x >= 0 and fmod(x, 1) == 0
```
