# \mathbb{C} — комплексные числа

---

## Синтаксис

```latex
x \in \mathbb{C}
x \in ℂ
```

---

## Трансляция

```c
1   // всегда true — mclang не имеет комплексного типа
```

В MVP комплексные числа не поддерживаются как тип. `x ∈ ℂ` — всегда истина.

---

## Пример

```mc
always_true(x) = x ∈ ℂ   // → 1
```
