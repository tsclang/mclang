# \frac{a}{b}

---

## Синтаксис

```latex
\frac{числитель}{знаменатель}
```

---

## Трансляция

```mc
\frac{a}{b}   →   a / b
```

```c
a / b
```

---

## Примеры

```mc
f(a, b) = \frac{a}{b}
g(x) = \frac{x^2 + 1}{x - 1}
h(x) = \frac{d}{dx} sin(x)    // это производная, не дробь
```

---

## Вложенные дроби

```mc
f(x) = \frac{\frac{1}{x}}{x + 1}   // → (1/x) / (x+1)
```
