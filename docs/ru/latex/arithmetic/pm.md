# \pm

---

## Синтаксис

```latex
\pm expr
```

---

## Трансляция

```mc
\pm expr   →   [+expr, -expr]   →   num[2]
```

---

## Примеры

```mc
roots(a, b, c) = (-b \pm \sqrt{b^2 - 4ac}) / (2a)
```
