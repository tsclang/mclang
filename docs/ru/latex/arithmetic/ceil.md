# \lceil x \rceil

---

## Синтаксис

```latex
\lceil x \rceil
```

---

## Трансляция

```mc
\lceil x \rceil   →   ceil(x)
```

---

## Примеры

```mc
f(x) = \lceil x / 2 \rceil
chunks(n, size) = \lceil n / size \rceil
```
