# \abs{x} / \lvert x \rvert

---

## Синтаксис

```latex
\abs{x}
\lvert x \rvert
|x|            // Unicode-синоним
```

---

## Трансляция

```mc
\abs{x}   →   abs(x)   →   fabs(x)
```

---

## Примеры

```mc
f(x) = \abs{x - 1}
dist(a, b) = \abs{a - b}
```
