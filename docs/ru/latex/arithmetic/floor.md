# \lfloor x \rfloor

---

## Синтаксис

```latex
\lfloor x \rfloor
```

---

## Трансляция

```mc
\lfloor x \rfloor   →   floor(x)
```

```c
floor(x)
```

---

## Примеры

```mc
f(x) = \lfloor x / 2 \rfloor
pages(n, per_page) = \lfloor n / per_page \rfloor
```
