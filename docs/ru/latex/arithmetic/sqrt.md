# \sqrt{x}

---

## Синтаксис

```latex
\sqrt{x}
```

---

## Трансляция

```mc
\sqrt{x}   →   sqrt(x)
```

```c
sqrt(x)
```

---

## Примеры

```mc
hyp(a, b) = \sqrt{a^2 + b^2}
unit(v: num[]) = v / \sqrt{dot(v, v)}
```
