# \left, \right — группировка выражений

`\left` и `\right` управляют размером скобок в LaTeX. В mclang они полностью игнорируются лексером — воспринимается только вложенная скобка.

## Поддерживаемые формы

| LaTeX | Эквивалент в mclang |
|-------|-------------------|
| `\left( expr \right)` | `(expr)` |
| `\left[ expr \right]` | `[expr]` (массив / индекс) |
| `\left\{ expr \right\}` | `{expr}` (аргумент команды) |
| `\left\| expr \right\|` | `\|expr\|` (норма вектора) |
| `\left\lvert expr \right\rvert` | `\|expr\|` (модуль / норма) |

## Пример

```mc
f(x) = \left( \frac{x+1}{x-1} \right)^2
// идентично: f(x) = ((x+1)/(x-1))^2
```

`\left` и `\right` не влияют на семантику — только на визуальный размер скобок в LaTeX-редакторе.
