# \sinh, \cosh, \tanh, \coth

---

## Синтаксис

```latex
\sinh{x}    \cosh{x}    \tanh{x}    \coth{x}
```

---

## Трансляция

| LaTeX | Си |
|-------|----|
| `\sinh{x}` | `sinh(x)` |
| `\cosh{x}` | `cosh(x)` |
| `\tanh{x}` | `tanh(x)` |
| `\coth{x}` | `cosh(x)/sinh(x)` |
