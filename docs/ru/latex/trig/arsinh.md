# \arsinh, \arcosh, \artanh

```latex
\arsinh{x}    \arcosh{x}    \artanh{x}
```

ISO-стандартные LaTeX-синонимы обратных гиперболических функций:

| LaTeX | mclang | Си |
|-------|--------|----|
| `\arsinh{x}` | `asinh(x)` | `asinh(x)` |
| `\arcosh{x}` | `acosh(x)` | `acosh(x)` |
| `\artanh{x}` | `atanh(x)` | `atanh(x)` |

Смотри [functions: asinh, acosh, atanh](../../functions/trig/asinh-acosh-atanh.md).
