# \sup_{x \in v} — максимум по массиву

```latex
\sup_{x \in v} expr
```

Транслируется в `max(v)` (максимальный элемент массива). В математике supremum совпадает с максимумом для конечного множества.

## Синтаксис

| Форма | Трансляция |
|-------|-----------|
| `\sup_{x \in v} x` | `max(v)` |
| `\sup_{x \in v} expr` | цикл с аккумулятором максимума |

## Трансляция

```mc
\sup_{x \in v} x
```

```c
mc_max(v, v_len)
```

## Пример

```mc
upper_bound(v: num[]) = \sup_{x \in v} x
```
