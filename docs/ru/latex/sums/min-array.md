# \min_{x \in v} — минимум по массиву

---

## Синтаксис

```latex
\min_{x \in v} expr
```

---

## Трансляция

```mc
\min_{x \in v} x^2
```

```c
mc_num _acc = v[0];
for (int _i = 1; _i < v_len; _i++) {
    mc_num x = v[_i];
    if (pow(x, 2.0) < _acc) _acc = pow(x, 2.0);
}
```

---

## Отличие от `min(v)`

`min(v)` возвращает минимальный **элемент** массива. `\min_{x \in v} expr` возвращает минимум произвольного **выражения** от каждого элемента.

```mc
// Минимальный элемент
a(v) = min(v)              // → fmin по элементам

// Минимум квадратов (другой результат!)
b(v) = \min_{x \in v} x^2
```

---

## Примеры

```mc
// Элемент, ближайший к нулю по квадрату
closest_to_zero(v: num[]) = \min_{x \in v} x^2

// Минимальное расстояние от точки p до элементов
min_dist(v: num[], p) = \min_{x \in v} (x - p)^2
```
