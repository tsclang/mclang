# \inf_{x \in v} — минимум по массиву

```latex
\inf_{x \in v} expr
```

Транслируется в `min(v)` (минимальный элемент массива). В математике infimum совпадает с минимумом для конечного множества.

## Синтаксис

| Форма | Трансляция |
|-------|-----------|
| `\inf_{x \in v} x` | `min(v)` |
| `\inf_{x \in v} expr` | цикл с аккумулятором минимума |

## Трансляция

```mc
\inf_{x \in v} x
```

```c
mc_min(v, v_len)
```

## Пример

```mc
lower_bound(v: num[]) = \inf_{x \in v} x
```
