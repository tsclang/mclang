# \sigma{v} — стандартное отклонение по массиву

```latex
\sigma{v}
```

Транслируется в вызов `std(v)`. Если за `\sigma` следует `{v}` — это функция; если `\sigma` стоит одиноко — греческая буква-идентификатор.

## Синтаксис

| Форма | Трансляция |
|-------|-----------|
| `\sigma{v}` | `std(v)` |
| `std(v)` | `std(v)` |
| `\sigma` (одиноко) | идентификатор `σ` |

## Трансляция

```mc
\sigma{v}
```

```c
mc_std(v, v_len)
```

## Пример

```mc
cv(v: num[]) = \sigma{v} / \bar{v}   // коэффициент вариации
```
