## Использование как идентификаторы

```mc
f(λ, μ) = λ * exp(-μ * t)       // λ, μ — параметры
maxwell(σ) = exp(-σ^2 / 2)      // σ — стандартное отклонение
lorentz(β) = 1 / sqrt(1 - β^2) // β = v/c
```

---

## Предостережения

`\pi`, `\phi` зарезервированы как константы — нельзя использовать как параметры:

```mc
f(\pi) = ...   // ошибка: π — встроенная константа
f(p) = p * π   // правильно: используй другое имя
```
