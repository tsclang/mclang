# Заглавные греческие буквы

---

## Таблица (только с LaTeX-синтаксисом)

| LaTeX | Символ | Транслитерация в Си |
|-------|--------|---------------------|
| `\Gamma` | Γ | `Gamma` (но `\Gamma{x}` = `tgamma(x)`) |
| `\Delta` | Δ | `Delta` |
| `\Theta` | Θ | `Theta` |
| `\Lambda` | Λ | `Lambda` |
| `\Xi` | Ξ | `Xi` |
| `\Pi` | Π | `Pi` |
| `\Sigma` | Σ | `Sigma` (но `\Sigma{v}` = `std(v)`) |
| `\Phi` | Φ | `Phi` |
| `\Psi` | Ψ | `Psi` |
| `\Omega` | Ω | `Omega` |

Буквы без LaTeX-команд (A, B, E, Z, H, I, K, M, N, O, P, T, Y, X) выглядят как латинские — используй Unicode-символы напрямую.

---

## Особые случаи

`\Gamma{x}` и `\Sigma{v}` воспринимаются как **функции**:

```mc
g(x) = \Gamma{x}   // → tgamma(x), не идентификатор Γ
s(v) = \Sigma{v}   // → std(v), не идентификатор Σ
```

Чтобы использовать как идентификаторы — пиши Unicode напрямую:

```mc
f(Γ, Σ) = Γ + Σ   // Γ и Σ — параметры
```
