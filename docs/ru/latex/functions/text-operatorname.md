# \text и \operatorname — вызов функций через LaTeX

`\text{name}` и `\operatorname{name}` позволяют вызвать произвольную функцию mclang, используя LaTeX-стиль именования.

---

## Синтаксис

```latex
\text{имя}(аргументы)
\operatorname{имя}(аргументы)
```

---

## Как работает

Лексер извлекает `имя` из фигурных скобок и генерирует токен `Identifier("имя")`. После этого парсер обрабатывает вызов как обычный вызов функции.

Это позволяет использовать в LaTeX-выражениях функции с произвольными именами:

```mc
f(x) = \text{my_func}(x) + \operatorname{clamp}(x, 0, 1)
```

Генерируется:

```c
mc_num f(mc_num x) {
    return my_func(x) + clamp(x, 0.0, 1.0);
}
```

---

## Применение

### Встроенные функции

```mc
g(x) = \text{floor}(x) + \text{ceil}(x)
```

### Пользовательские функции

```mc
_helper(x) = x * 2

f(x) = \operatorname{_helper}(x)
```

### Математические операторы в LaTeX-стиле

В LaTeX принято писать операторы через `\operatorname{}`:

```latex
\operatorname{sgn}(x)   // sign function
\operatorname{erf}(x)   // error function
```

Эти записи полностью поддерживаются в mclang.

---

## Отличие от \sigma и \Gamma

`\sigma{v}` и `\Gamma{x}` — специальные случаи: они распознаются лексером как вызов конкретных функций (`std` и `tgamma`) только если за ними следует `{`. Без `{}` они становятся идентификаторами греческих букв.

`\text{name}` и `\operatorname{name}` — универсальный механизм для любого имени.
