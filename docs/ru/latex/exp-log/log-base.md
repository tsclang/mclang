# \log_{b}{x}

---

## Синтаксис

```latex
\log_{b}{x}
\log_{2}{x}    // log₂
\log_{10}{x}   // = \lg
```

---

## Трансляция

```mc
\log_{b}{x}   →   log(b, x)   →   log(x)/log(b)
```

---

## Примеры

```mc
bits(n) = \log_{2}{n}
ph(c) = -\log_{10}{c}
```
