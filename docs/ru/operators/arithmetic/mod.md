# Оператор % (остаток)

---

## Синтаксис

```mc
a % b
a mod b     // словесный синоним
mod(a, b)   // функциональный синоним
\mod        // LaTeX (в контексте выражения)
```

---

## Пример

```mc
remainder(a, b) = a % b
```

```c
mc_num remainder(mc_num a, mc_num b) { return fmod(a, b); }
```

Генерируется `fmod` из `<math.h>` — корректно работает с дробными числами.

---

## Отличие от целочисленного %

`fmod` сохраняет знак делимого (как в Си `%` для целых):

```mc
f(a, b) = a mod b

// f(7, 3)   → 1
// f(-7, 3)  → -1   (знак от делимого)
// f(7, -3)  → 1
// f(7.5, 2) → 1.5  // работает с дробными
```

---

## Деление по модулю (всегда положительный)

```mc
mod_positive(a, b) = ((a % b) + b) % b
```

---

## Проверка чётности

```mc
is_even(n) = n % 2 == 0
is_odd(n)  = n % 2 != 0
```

---

## Ошибки

### Остаток от деления на ноль

```mc
f(x) = x % 0   // → nan (fmod(x, 0))
```
