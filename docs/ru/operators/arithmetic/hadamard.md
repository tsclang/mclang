# Оператор .* (Hadamard-произведение)

Поэлементное умножение двух массивов одинаковой длины.

---

## Синтаксис

```mc
a .* b
```

---

## Семантика

`a .* b` умножает каждый элемент массива `a` на соответствующий элемент `b`:

```
result[i] = a[i] * b[i]
```

Оба массива должны иметь одинаковую длину. Длина результата равна длине операндов.

---

## Пример

```mc
weight(a: num[], w: num[]) = a .* w   // взвешенные значения
```

```c
mc_num* weight(const mc_num* a, int a_len, const mc_num* w, int w_len) {
    return mc_mul_arr(a, w, a_len);
}
```

---

## Отличие от ⋅ (точечное произведение)

| Оператор | Операнды | Результат |
|----------|----------|-----------|
| `a .* b` | `num[]`, `num[]` | `num[]` — поэлементно |
| `a ⋅ b`  | `num[]`, `num[]` | `num` — скалярное произведение |
| `A ⋅ B`  | `num[][]`, `num[][]` | `num[][]` — матричное умножение |

---

## LaTeX

```latex
a \cdot b    // ⋅ — dot product или матричное умножение (зависит от типа)
```

Нотация `.*` специфична для mclang и не имеет стандартного LaTeX-эквивалента.
