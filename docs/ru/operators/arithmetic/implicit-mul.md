# Неявное умножение

mclang поддерживает математическую запись без явного `*` — число перед переменной, скобкой или функцией воспринимается как умножение.

---

## Правила

Неявное умножение применяется когда:
- Число стоит перед идентификатором: `2x`, `3π`
- Число стоит перед скобкой: `2(a+b)`
- Идентификатор стоит перед скобкой: `x(y+1)` — **только если x не функция**
- Скобка перед скобкой: `(a+b)(c+d)`

---

## Примеры

```mc
f(x)    = 2x        // 2 * x
g(r)    = 2πr       // 2 * π * r
h(a, b) = 2(a + b)  // 2 * (a + b)
k(a, b) = (a+1)(b+1)// (a+1) * (b+1)
```

```c
mc_num f(mc_num x)           { return 2.0 * x; }
mc_num g(mc_num r)           { return 2.0 * M_PI * r; }
mc_num h(mc_num a, mc_num b) { return 2.0 * (a + b); }
```

---

## Физические формулы

```mc
kinetic_energy(m, v) = (1/2)m v^2
wave_number(f, c) = 2πf / c
lorentz(β) = 1 / sqrt(1 - β^2)
```

---

## Приоритет неявного умножения

Неявное умножение имеет тот же приоритет, что и явное `*`:

```mc
f(x) = 2x^2    // → 2 * (x^2), не (2x)^2
g(x) = 1/2x    // → (1/2) * x = 0.5x
```

---

## Что не является неявным умножением

```mc
sin(x)     // вызов функции, не sin * x
arr[i]     // индексация, не arr * i
f.length   // свойство, не f * length
```

---

## Ошибки

### Неоднозначность: функция или умножение

```mc
f = 2
g(x) = f(x)   // вызов f как функции или f * x?
```

Если `f` объявлена как функция — вызов. Если как константа — умножение. При неоднозначности компилятор выдаёт предупреждение.
