# Оператор / (деление)

---

## Синтаксис

```mc
a / b
a ÷ b   // синоним
```

---

## Скалярное деление

```mc
f(a, b) = a / b
```

```c
mc_num f(mc_num a, mc_num b) { return a / b; }
```

---

## Деление на ноль

Деление на ноль не вызывает ошибки — возвращает `inf`, `-inf` или `nan` по правилам IEEE 754:

```mc
f(x) = 1 / x
// f(0)   → inf
// f(-0)  → -inf
// f(inf) → 0
// 0/0    → nan
```

---

## LaTeX-деление: `\frac`

```mc
f(a, b) = \frac{a}{b}
```

Эквивалентно `a / b`. Генерирует тот же Си-код.

---

## Целочисленное деление

mclang не имеет оператора целочисленного деления. Используй `floor`:

```mc
int_div(a, b) = floor(a / b)
```

---

## Деление вектора на скаляр

```mc
normalize_scale(v: num[], s) = v / s
```

---

## Ошибки

### Деление вектора на вектор

```mc
a: num[]
b: num[]
r = a / b   // ошибка: используй .* с обратными элементами
```

**Исправление** — поэлементное:

```mc
r = a .* (1 / b)   // если реализовано поэлементное деление
```
