# Оператор - (вычитание и унарный минус)

---

## Синтаксис

```mc
a - b    // бинарный
-a       // унарный
```

---

## Бинарное вычитание

```mc
diff(x, y) = x - y
```

```c
mc_num diff(mc_num x, mc_num y) { return x - y; }
```

---

## Унарный минус

```mc
neg(x) = -x
```

```c
mc_num neg(mc_num x) { return -x; }
```

---

## Со встроенными константами

```mc
f(x) = x - π
g(x) = -inf      // отрицательная бесконечность
```

---

## Векторное вычитание (поэлементно)

```mc
sub_vec(a: num[], b: num[]) = a - b
```

---

## ASI и унарный минус

Осторожно с переносом строк — унарный минус может быть принят за новое выражение:

```mc
// Плохо
result = a
    - b     // парсер видит: result = a; затем -b (отдельное выражение)

// Хорошо
result = a -
    b
```
