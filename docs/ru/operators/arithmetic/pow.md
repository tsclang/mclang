# Оператор ^ (степень)

---

## Синтаксис

```mc
x^n
x**n     // синоним
x^{n}    // LaTeX-форма
```

---

## Базовые примеры

```mc
square(x) = x^2
cube(x) = x^3
```

```c
mc_num square(mc_num x) { return pow(x, 2.0); }
mc_num cube(mc_num x) { return pow(x, 3.0); }
```

---

## Правоассоциативность

```mc
f(x) = 2^3^2   // → 2^(3^2) = 2^9 = 512, не (2^3)^2 = 64
```

---

## e^x

Компилятор автоматически оптимизирует `e^x` → `exp(x)`:

```mc
f(x) = e^x
```

```c
mc_num f(mc_num x) { return exp(x); }
```

---

## Дробная степень

```mc
f(x) = x^(1/3)   // кубический корень
g(x) = x^0.5     // квадратный корень (хуже читается, используй sqrt)
```

---

## LaTeX: `x^{n}` и `x^{-1}`

```mc
inverse(x) = x^{-1}    // → 1/x
f(x) = (x+1)^{2}       // → pow(x+1, 2)
```

---

## Степень матрицы

Нет встроенного оператора степени для матриц. Для `A^2` используй `A ⋅ A`.

---

## Ошибки

### Степень отрицательного числа с дробным показателем

```mc
f(x) = (-2)^(1/2)   // → nan (комплексное число)
```

Используй `abs` если нужен модуль:

```mc
f(x) = abs(x)^(1/2)
```
