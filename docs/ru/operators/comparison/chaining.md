# Цепочки сравнений

mclang поддерживает математическую запись цепочек сравнений вида `a < b < c`, которая разворачивается в `(a < b) and (b < c)`.

---

## Синтаксис

```mc
a < b < c
a <= b <= c
a < b <= c
lo <= x < hi
```

---

## Разворачивание

```mc
in_open(x, lo, hi) = lo < x < hi
```

```c
mc_num in_open(mc_num x, mc_num lo, mc_num hi) {
    return ((lo < x) && (x < hi)) ? 1.0 : 0.0;
}
```

Каждый промежуточный операнд вычисляется **один раз**.

---

## Все поддерживаемые операторы в цепочке

`<`, `<=`, `>`, `>=`, `==`, `!=`

```mc
ascending(a, b, c, d) = a < b < c < d
bounded(x) = -1 <= x <= 1
```

---

## Примеры применения

```mc
// Проверка единичного интервала
unit(x) = 0 <= x <= 1

// Открытый интервал
open_range(x, a, b) = a < x < b

// Полуоткрытый
half_open(x, a, b) = a <= x < b
```

---

## Ограничения

Цепочки смешанного направления (`a < b > c`) допустимы синтаксически, но математически некорректны в большинстве случаев:

```mc
// Допустимо, но означает: (a < b) and (b > c)
f(a, b, c) = a < b > c
```
