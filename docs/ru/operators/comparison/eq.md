# Оператор == (равенство)

---

## Синтаксис

```mc
a == b
```

Возвращает `1.0` (истина) или `0.0` (ложь) типа `num`.

---

## Пример

```mc
is_zero(x) = x == 0
is_pi(x)   = x == π
```

```c
mc_num is_zero(mc_num x) { return (x == 0.0) ? 1.0 : 0.0; }
```

---

## В условиях if

```mc
f(x) =
    if (x == 0) inf
    1 / x
```

---

## Сравнение с плавающей точкой

Прямое сравнение `==` для `num` может давать неожиданные результаты из-за ошибок округления:

```mc
// Потенциально неточно:
f(x) = x == 0.1 + 0.2   // может вернуть 0 даже если x = 0.3
```

**Рекомендация** — сравнивать с эпсилоном:

```mc
approx_eq(a, b, eps) = abs(a - b) < eps
```

---

## nan не равен ни чему

```mc
// nan != nan всегда (IEEE 754)
check(x) = x == nan   // всегда 0 (false)
```

Используй `is_nan(x)` для проверки.
