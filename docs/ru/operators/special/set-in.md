# Оператор ∈ (принадлежность множеству)

---

## Синтаксис

```mc
x ∈ S
x \in S   // LaTeX
x in S    // словесный синоним
```

---

## Поддерживаемые множества

| Запись | Множество | Условие в Си |
|--------|-----------|-------------|
| `x ∈ ℕ` | натуральные числа | `x >= 0 && fmod(x,1) == 0` |
| `x ∈ ℤ` | целые числа | `fmod(x,1) == 0` |
| `x ∈ ℝ` | вещественные числа | `isfinite(x)` |
| `x ∈ ℚ` | рациональные | `isfinite(x)` |
| `x ∈ ℂ` | комплексные | `1` (всегда true) |

---

## Интервалы

```mc
x ∈ [a, b]    // замкнутый: a <= x <= b
x ∈ (a, b)    // открытый: a < x < b
x ∈ [a, b)    // полуоткрытый: a <= x < b
x ∈ (a, b]    // полуоткрытый: a < x <= b
```

---

## Примеры

```mc
is_natural(x)  = x ∈ ℕ
is_integer(x)  = x ∈ ℤ
is_real(x)     = x ∈ ℝ

in_unit(x)     = x ∈ [0, 1]
in_open(x)     = x ∈ (0, 1)
```

Генерируемый Си-код:

```c
mc_num is_natural(mc_num x) {
    return ((x >= 0.0) && (fmod(x, 1.0) == 0.0)) ? 1.0 : 0.0;
}

mc_num in_unit(mc_num x) {
    return ((x >= 0.0) && (x <= 1.0)) ? 1.0 : 0.0;
}
```

---

## В guard-условиях

```mc
probability(p) =
    p
    where p ∈ [0, 1]   // guard: p должна быть вероятностью
```
