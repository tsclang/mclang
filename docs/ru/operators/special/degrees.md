# Оператор ° (градусы)

Постфиксный оператор `°` автоматически переводит градусы в радианы.

---

## Синтаксис

```mc
x°
```

---

## Пример

```mc
sin_deg(x) = sin(x°)
```

```c
mc_num sin_deg(mc_num x) { return sin(x * (M_PI / 180.0)); }
```

---

## Генерируемый Си-код

`x°` всегда генерирует:

```c
(x * (M_PI / 180.0))
```

---

## Типичное использование

```mc
// Угол 45°
f = sin(45°)     // → sin(π/4) ≈ 0.7071

// Угол как параметр
rotate(angle_deg) =
    cos(angle_deg°)
```

---

## Ввод символа °

| Система | Способ |
|---------|--------|
| Windows | `Alt + 0176` (numpad) |
| macOS | `Opt + Shift + 8` |
| Linux | `Compose + d + g` или `Ctrl+Shift+U 00B0 Enter` |
| VS Code | вставка Unicode |

---

## Ошибки

### Двойное применение

```mc
f(x) = sin(x°°)   // ошибка: ° применяется дважды
```
