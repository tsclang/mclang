# Интеграция с Rust

mclang компилирует `.mc`-файл в `.c` + `.h` + `_bindings.rs` — готовые FFI-биндинги для Rust.

---

## Быстрый старт

### 1. Скомпилировать с таргетом rust

```bash
mclang physics.mc --target rust
# → physics.c
# → physics.h
# → physics_bindings.rs
```

### 2. Настроить Cargo.toml

```toml
[build-dependencies]
cc = "1"
```

### 3. Создать build.rs

```rust
fn main() {
    cc::Build::new()
        .file("physics.c")
        .flag("-lm")
        .compile("physics");
}
```

### 4. Подключить биндинги

```rust
include!("../physics_bindings.rs");

fn main() {
    let result = unsafe { range(50.0, 0.785) };
    println!("range = {result}");
}
```

## Пример: баллистика

**ballistics.mc:**
```mc
range(v0, angle) = v0^2 * sin(2*angle) / 9.81
max_height(v0, angle) = v0^2 * sin(angle)^2 / (2*9.81)
time_of_flight(v0, angle) = 2 * v0 * sin(angle) / 9.81
```

```bash
mclang ballistics.mc --target rust
```

**ballistics_bindings.rs** (генерируется автоматически):
```rust
// Auto-generated by mclang — do not edit
//
// Cargo.toml:
//   [build-dependencies]
//   cc = "1"
//
// build.rs:
//   fn main() {
//       cc::Build::new().file("ballistics.c").compile("ballistics");
//   }

#[allow(non_snake_case, dead_code)]
unsafe extern "C" {
    pub fn range(v0: f64, angle: f64) -> f64;
    pub fn max_height(v0: f64, angle: f64) -> f64;
    pub fn time_of_flight(v0: f64, angle: f64) -> f64;
}
```

**main.rs:**
```rust
include!("ballistics_bindings.rs");

fn main() {
    let v0 = 50.0_f64;
    let angle = 45_f64.to_radians();
    unsafe {
        println!("range          = {:.2}", range(v0, angle));
        println!("max_height     = {:.2}", max_height(v0, angle));
        println!("time_of_flight = {:.2}", time_of_flight(v0, angle));
    }
}
```

## Маппинг типов

| mclang | Rust |
|--------|------|
| `num` (f64) | `f64` |
| `num` (f32) | `f32` |
| `num` (fixed) | `i16` |
| `int` | `i32` |
| `num[]` | `*const f64, i32` (ptr + len) |
| `num[][]` | `*const f64, i32, i32` (ptr + rows + cols) |

### Передача массива

```rust
let v = vec![1.0_f64, 2.0, 3.0];
let n = unsafe { norm_sq(v.as_ptr(), v.len() as i32) };
```

## Функции с параметрами по умолчанию

```mc
f(x, y = 0.0) = x + y
```

Генерирует два варианта:

```rust
unsafe extern "C" {
    // все параметры явно
    pub fn _f_impl(x: f64, y: f64) -> f64;
    // y заполняется дефолтом (0.0)
    pub fn _f_d1(x: f64) -> f64;
}
```

Использование:
```rust
unsafe {
    let a = _f_d1(5.0);        // y = 0.0 по умолчанию → 5.0
    let b = _f_impl(5.0, 3.0); // y = 3.0 явно → 8.0
}
```
