# Значимые отступы

mclang использует значимые отступы в стиле Python. Блок начинается после `:` (или `=` для однострочных функций) и заканчивается при уменьшении уровня отступа.

---

## Правила

- Отступ должен быть **последовательным** внутри одного блока (не смешивай пробелы и табы)
- Стандарт: **4 пробела** или **1 таб** на уровень
- Пустые строки и строки с одними пробелами внутри блока допустимы
- Комментарии не влияют на уровень отступа

---

## Однострочная функция

```mc
f(x) = x^2 + 1
```

Тело — одно выражение, без отступа.

---

## Многострочная функция

Тело начинается с увеличения отступа после `=`:

```mc
clamp(x, lo, hi) =
    if (x < lo) lo
    else if (x > hi) hi
    else x
```

---

## Блок `where`

`where` объявляется на уровне функции, его тело — с отступом:

```mc
trajectory(v0, angle, t) =
    x + y
    where
        g  = 9.81
        x  = v0 * cos(angle) * t
        y  = v0 * sin(angle) * t - 0.5 * g * t^2
```

---

## Вложенные блоки

```mc
f(n) =
    if (n <= 0)
        0
    else
        for i in 1..n
            acc = acc + i
        acc
    where
        acc = 0
```

---

## Ошибки

### Непоследовательный отступ

```mc
f(x) =
    a = 1
      b = 2    // ошибка: другой уровень отступа
```

**Исправление:** выровняй `b = 2` на тот же уровень, что `a = 1`.

### Смешение пробелов и табов

```mc
f(x) =
	a = 1      // таб
    b = 2      // 4 пробела  → ошибка
```

**Исправление:** используй только один тип пробельных символов во всём файле.
