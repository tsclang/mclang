# Объявление функций

Функция в mclang — это именованное выражение. Объявляется в формате `имя(параметры) = тело`.

---

## Синтаксис

```
имя(параметры) = выражение
```

```mc
circle_area(r) = π * r^2
```

---

## Параметры

### Без типа

```mc
f(x) = x^2
```

Тип выводится как `num` (double по умолчанию).

### С типом

```mc
f(x: num) = x^2
factorial(n: int) = n!
dot_product(a: num[], b: num[]) = dot(a, b)
```

Доступные типы: `num`, `int`, `num[]`, `num[][]`.

### Значения по умолчанию

```mc
power(x, n = 2) = x^n
```

Вызов: `power(3)` → `9.0`, `power(3, 3)` → `27.0`.

Параметры с дефолтом должны стоять **после** обязательных:

```mc
// Правильно
f(x, y = 0, z = 0) = x + y + z

// Ошибка: обязательный параметр после параметра с дефолтом
f(x = 0, y) = x + y   // → ошибка компиляции
```

---

## Экспорт

Все функции экспортируются по умолчанию — попадают в `.h` файл:

```c
// hello.h
mc_num circle_area(mc_num r);
```

Исключение: функции с `_` в начале имени (см. [Приватные функции](private.md)).

---

## Несколько функций

```mc
// Файл geometry.mc

circle_area(r) = π * r^2
circle_perimeter(r) = 2 * π * r

rect_area(w, h) = w * h
rect_perimeter(w, h) = 2 * (w + h)
```

---

## Глобальные константы

На верхнем уровне можно объявлять именованные константы:

```mc
G = 6.674e-11
c = 2.998e8

gravitational_force(m1, m2, r) = G * m1 * m2 / r^2
```

Подробнее: [Глобальные константы](../constants/global.md).

## Генерируемый Си-код

Для функции:

```mc
circle_area(r) = π * r^2
```

Генерируется:

```c
// circle.h
mc_num circle_area(mc_num r);

// circle.c
mc_num circle_area(mc_num r) {
    return (M_PI * pow(r, 2.0));
}
```

---

## Ошибки

### Дублирование имени

```mc
f(x) = x + 1
f(x) = x * 2   // ошибка: f уже объявлена
```

### Неизвестный параметр в выражении

```mc
f(x) = x + y   // ошибка: y не объявлен
```

**Исправление:** добавь `y` как параметр или в блок `where`:

```mc
f(x) = x + y
    where y = 2.0
```
