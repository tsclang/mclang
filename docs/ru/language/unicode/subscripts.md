# Нижние индексы

mclang поддерживает Unicode-символы нижних индексов (U+2080–U+2089) в идентификаторах.

---

## Символы

| Символ | Unicode | Транслитерация |
|--------|---------|----------------|
| `₀` | U+2080 | `_0` |
| `₁` | U+2081 | `_1` |
| `₂` | U+2082 | `_2` |
| `₃` | U+2083 | `_3` |
| `₄` | U+2084 | `_4` |
| `₅` | U+2085 | `_5` |
| `₆` | U+2086 | `_6` |
| `₇` | U+2087 | `_7` |
| `₈` | U+2088 | `_8` |
| `₉` | U+2089 | `_9` |

---

## Пример

```mc
// Начальные условия
x₀ = 0.0
v₀ = 10.0
a₀ = -9.81

position(t) = x₀ + v₀ * t + 0.5 * a₀ * t^2
```

```c
static const mc_num x_0 = 0.0;
static const mc_num v_0 = 10.0;
static const mc_num a_0 = -9.81;

mc_num position(mc_num t) {
    return x_0 + v_0 * t + 0.5 * a_0 * pow(t, 2.0);
}
```

---

## Комбинации с греческими буквами

```mc
ω₀ = 2 * π * 50   // угловая частота 50 Гц
φ₀ = π / 4        // начальная фаза

oscillation(t) = cos(ω₀ * t + φ₀)
```

```c
mc_num oscillation(mc_num t) {
    return cos((6.283185... * t) + 0.785398...);
}
```

---

## LaTeX-альтернатива

Если редактор не поддерживает ввод нижних индексов, можно использовать LaTeX-синтаксис:

```mc
// x_{0} — эквивалентно x₀
position(t) = x_{0} + v_{0} * t
```

---

## Ввод в редакторе

- **VS Code**: используй плагин Unicode Input или копируй из таблицы символов
- **JetBrains**: `View → Insert Unicode Character`
- **Любой редактор**: вставка через буфер обмена

---

## Ограничения

- Только цифровые нижние индексы ₀–₉ — буквенные нижние индексы (ₐ, ₑ и т.д.) не поддерживаются
- Нижний индекс не может быть первым символом идентификатора: `₀x` — ошибка
