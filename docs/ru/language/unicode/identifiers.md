# Unicode-идентификаторы

mclang поддерживает Unicode в именах функций, параметров и переменных. Греческие буквы и математические символы можно использовать напрямую.

---

## Греческие буквы

Все буквы греческого алфавита допустимы как идентификаторы:

```mc
λ(x) = e^(-λ * x)       // λ — и имя функции, и параметр

angular_velocity(θ, t) = θ / t

maxwell(v, T) =
    4 * π * (m / (2 * π * k * T))^(3/2) * v^2 * e^(-m * v^2 / (2 * k * T))
    where
        m = 1.67e-27
        k = 1.38e-23
```

## Транслитерация в Си

В сгенерированном Си-коде Unicode-идентификаторы транслитерируются в ASCII:

```mc
angular_force(θ, ω) = θ * ω
```

```c
mc_num angular_force(mc_num theta, mc_num omega) {
    return theta * omega;
}
```

Правила транслитерации: [Транслитерация](transliteration.md).

---

## Индексированные переменные

```mc
x₁ = 1.0
x₂ = 2.0
distance(x₁, x₂) = abs(x₁ - x₂)
```

Нижние индексы (`₁`, `₂`) допустимы в идентификаторах.

---

## Ограничения

- Эмодзи и пиктограммы — не допускаются
- Имя не может начинаться с цифры или символа (кроме `_`)
- Символы `π`, `e`, `τ`, `φ`, `inf`, `nan` зарезервированы как константы — нельзя использовать как имена переменных

---

## Ошибки

### Перекрытие встроенной константы

```mc
f(π) = π * 2   // ошибка: π — встроенная константа, нельзя использовать как параметр
```

**Исправление:**

```mc
f(r) = π * r^2
```
