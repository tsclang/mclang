# Тип num

`num` — универсальный числовой тип mclang. Он адаптируется под выбранный таргет компиляции.

---

## Соответствие в Си

| Флаг компиляции | Си-тип | Размер |
|-----------------|--------|--------|
| `--precision f64` (по умолчанию) | `double` | 64 бит |
| `--precision f32` | `float` | 32 бит |
| `--precision fixed` | `int16_t` | 16 бит, Q8.8 |

В Си-коде тип всегда называется `mc_num`:

```c
typedef double mc_num;   // при --precision f64
```

---

## Объявление параметра

```mc
f(x: num) = x^2
```

Тип можно опустить — по умолчанию параметры имеют тип `num`:

```mc
f(x) = x^2   // эквивалентно f(x: num) = x^2
```

---

## Литералы

```mc
a = 3.14
b = 1.0
c = 6.674e-11
d = -0.5
```

Компилятор всегда записывает числа с точкой, чтобы избежать целочисленного деления в Си:

```c
mc_num a = 3.14;
mc_num b = 1.0;
```

---

## Специальные значения

| mclang | Си | Значение |
|--------|-----|---------|
| `inf` | `INFINITY` | +∞ |
| `-inf` | `-INFINITY` | -∞ |
| `nan` | `NAN` | не-число |

```mc
safe_div(a, b) =
    if (b == 0) inf
    a / b
```

---

## Проверка состояния

```mc
is_nan(x)      // isnan(x)
is_inf(x)      // isinf(x)
is_finite(x)   // isfinite(x)
```

---

## Деление на ноль

Деление на ноль возвращает `inf` или `nan`, не вызывает падения программы:

```mc
f(x) = 1 / x
// f(0) → inf (IEEE 754)
```

---

## Преобразование в int

Явного приведения нет — используй `floor`, `ceil`, `round`:

```mc
n: int = floor(x)
```
