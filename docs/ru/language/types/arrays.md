# Тип num[]

`num[]` — одномерный массив (вектор) чисел. При передаче в функцию автоматически добавляется неявный параметр длины `_len`.

---

## Объявление параметра

```mc
sum_arr(v: num[]) = sum(v)
```

Генерируемая Си-сигнатура:

```c
mc_num sum_arr(mc_num* v, int v_len);
```

Параметр `v_len` добавляется автоматически — его не нужно объявлять в mclang.

---

## Свойство `.length`

```mc
mean(v: num[]) = sum(v) / v.length
```

В Си: `v_len`.

---

## Литерал массива

```mc
v = [1.0, 2.0, 3.0, 4.0]
```

Генерируется:

```c
static mc_num _arr[] = {1.0, 2.0, 3.0, 4.0};
```

---

## Статический массив num[N]

Размер указывается явно — размещается на стеке:

```mc
buf: num[256]
```

Генерируется:

```c
mc_num buf[256];
```

---

## Доступ по индексу

```mc
first(v: num[]) = v[0]
last(v: num[]) = v[v.length - 1]
```

Индекс должен иметь тип `int`. Нет проверки выхода за границы — ответственность на пользователе.

---

## Срез

```mc
partial_sum(v: num[], lo: int, hi: int) =
    acc = 0
    for i in lo..hi
        acc = acc + v[i]
    acc
```

Диапазонные срезы `v[a..b]` создают временный указатель:

```mc
head(v: num[]) = v[0..4]   // первые 4 элемента
```

---

## Поэлементные операции

```mc
add_vectors(a: num[], b: num[]) = a + b    // поэлементное сложение
scale(v: num[], s: num) = s * v            // умножение на скаляр
hadamard(a: num[], b: num[]) = a .* b      // поэлементное умножение
```

---

## Встроенные функции для векторов

| Функция | Описание |
|---------|----------|
| `sum(v)` | сумма элементов |
| `mean(v)` | среднее |
| `std(v)` | стандартное отклонение |
| `min(v)` | минимум |
| `max(v)` | максимум |
| `norm(v)` | евклидова норма |
| `dot(a, b)` | скалярное произведение |

---

## Передача из Си

```c
double data[] = {1.0, 2.0, 3.0, 5.0, 8.0};
double result = sum_arr(data, 5);
```

---

## Ошибки

### Использование .length на матрице

```mc
m: num[][]
n = m.length   // ошибка: у матриц нет .length, используй .rows и .cols
```

**Исправление:**

```mc
n = m.rows
```

### Несовпадение длин в поэлементной операции

```mc
a = [1.0, 2.0, 3.0]
b = [1.0, 2.0]
c = a + b   // ошибка времени выполнения: разные длины
```
