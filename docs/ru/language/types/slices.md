# Срезы

Срезы позволяют обращаться к части вектора или к строке/столбцу матрицы без копирования данных.

---

## Срез вектора

```mc
// v[a..b] — элементы с индекса a по b включительно
head5(v: num[]) = v[0..4]
tail(v: num[], n: int) = v[v.length - n .. v.length - 1]
```

Генерируется указатель на нужную позицию:

```c
mc_num* _slice = v + 0;
int _slice_len = 5;
```

---

## Строка матрицы: `m[i, :]`

```mc
get_row(m: num[][], i: int) = m[i, :]
```

Возвращает i-ю строку как вектор. В Си — указатель `m + i * m_cols`:

```c
mc_num* _row = m + i * m_cols;
int _row_len = m_cols;
```

---

## Столбец матрицы: `m[:, j]`

```mc
get_col(m: num[][], j: int) = m[:, j]
```

Создаёт временный массив (столбец не хранится непрерывно):

```c
static mc_num _col[m_rows];
for (int _i = 0; _i < m_rows; _i++) _col[_i] = m[_i * m_cols + j];
```

---

## Применение

### Скалярное произведение строки на вектор

```mc
row_dot(m: num[][], i: int, v: num[]) =
    dot(m[i, :], v)
```

### Сумма столбца

```mc
col_sum(m: num[][], j: int) =
    sum(m[:, j])
```

### Диагональ

```mc
diag_sum(m: num[][]) =
    acc = 0
    for i in 0..m.rows - 1
        acc = acc + m[i, :][i]
    acc
```

---

## Ограничения

- Срез вектора `v[a..b]` — не копируется, это указатель. Изменение элементов среза изменяет оригинал
- Столбец `m[:, j]` — копируется во временный буфер `static`. Не используй два столбца одновременно
- Нет синтаксиса `m[a..b, c..d]` (прямоугольный подмассив) — в v2+

---

## Ошибки

### Инвертированный диапазон

```mc
v[5..2]   // ошибка: начало > конец
```

### Выход за границы

```mc
v = [1.0, 2.0, 3.0]
x = v[0..5]   // выход за границы — undefined behavior в Си
```
