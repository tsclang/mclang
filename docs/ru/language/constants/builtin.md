# Встроенные константы

mclang предоставляет набор математических констант, доступных в любом файле без объявления.

---

## Таблица констант

| mclang | Значение | Си-код | Описание |
|--------|----------|--------|----------|
| `π` или `pi` | 3.14159265358979… | `M_PI` | Число π |
| `e` | 2.71828182845904… | `M_E` | Основание натурального логарифма |
| `τ` или `tau` | 6.28318530717958… | `(2.0 * M_PI)` | Тау = 2π |
| `φ` или `phi` | 1.61803398874989… | `((1.0 + sqrt(5.0)) / 2.0)` | Золотое сечение |
| `inf` | +∞ | `INFINITY` | Положительная бесконечность |
| `-inf` | -∞ | `-INFINITY` | Отрицательная бесконечность |
| `nan` | NaN | `NAN` | Не-число |

---

## Примеры

```mc
circle_area(r) = π * r^2
circle_perimeter(r) = τ * r

golden_rect(a) = a * φ

safe_inv(x) =
    if (x == 0) inf
    1 / x
```

---

## e как основание степени

```mc
// e^x
exp_x(x) = e^x
```

Компилятор распознаёт `e^x` и генерирует `exp(x)`:

```c
mc_num exp_x(mc_num x) {
    return exp(x);
}
```

---

## Приоритет имён

Если пользователь объявляет константу с тем же именем — она перекрывает встроенную:

```mc
e = 2.718     // перекрывает встроенную e (не рекомендуется)
```

Компилятор выдаст предупреждение, но не ошибку.

## nan и деление на ноль

```mc
f(x) =
    if (x == 0) nan
    1 / x
```

`nan` можно проверить только через `is_nan(x)`, оператор `==` с nan всегда возвращает false (IEEE 754):

```mc
check(x) = is_nan(x)   // правильно
bad(x)   = x == nan    // всегда false — неверно
```
