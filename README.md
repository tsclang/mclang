# mclang (Math C Language)

mclang — это декларативный, встраиваемый язык-вычислитель, созданный для того, чтобы превращать сложные математические, физические и химические формулы в максимально производительный код на языке C.

## Основная идея

В отличие от языков общего назначения, mclang — это «чистое вычислительное ядро». В нем нет строк, ввода-вывода или сложного рантайма. Вы пишете формулы так, как они выглядят в учебниках или научной литературе, а компилятор генерирует автономный Си-код, готовый к интеграции в игры (физические движки), аналитические системы на Python или высокопроизводительные веб-приложения (через WebAssembly).

## Ключевые особенности

* Декларативный «тетрадный» синтаксис: Определение функций через равенства (f(x) = ...), отсутствие ключевых слов return и function, поддержка значимых отступов (Python-style).
* Математика уровня CAS: Прямая поддержка синтаксиса LaTeX (\frac, \sqrt, \int, \lim), неявное умножение (2πx),Unicode-символы переменных (α, β, γ) и цепочки сравнений (0 < x < 100).
* Интеллектуальная типизация (Zero-Types): Математику не нужно думать о битах. Основной тип данных — num. Компилятор сам подберет оптимальную точность (f32/f64/fixed-point) в зависимости от целевого железа (от 8-битных Arduino до современных CPU).
* Аналитический движок (McSimplify): Компилятор не просто транслирует код, а упрощает его на этапе сборки. Он умеет раскрывать скобки, сокращать дроби и использовать табличные интегралы и производные, чтобы итоговый Си-код работал молниеносно.
* Реактивный вывод (from): Возможность выражать переменные из готовых формул. Вы указываете эталонную формулу и искомую переменную, а mclang сам выводит математическое решение.
* Встроенная научная база: В язык «из коробки» вшиты константы Вселенной (G, c, h), периодическая таблица Менделеева и стандартные физические законы.

## Технический стек

* Backend: Чистый Си (стандарт C99/C11).
* Memory Management: Стэковая память, отсутствие динамической аллокации (zero-malloc), безопасность и предсказуемость.
* Interoperability: Автоматическая генерация биндингов для JavaScript (WebAssembly), Python (ctypes/C-API) и C/C++.
